Panduan Penggunaan Aplikasi

1. Hak Akses (Login)

  • Koordinator (Guru/Korlas): Memiliki akses penuh (Tambah Data, Absensi, Penilaian, Laporan).
  • User Umum (Wali Murid): Hanya dapat melihat menu Rekap & Laporan serta Profil Pemain untuk memantau kehadiran.

2. Mengelola Data Pemain

Masuk ke menu Data Pemain. Anda bisa:

  • Tambah Manual: Isi form nama, kelas, posisi, dll.
  • Upload Foto: Bisa upload file langsung dari HP/PC (disimpan lokal) atau menggunakan Link Google Drive agar lebih ringan.
  • Import Excel: Download template, isi data banyak siswa, lalu upload kembali.

3. Cara Menggunakan Foto Google Drive

  1. Upload foto anak ke Google Drive Anda.
  2. Klik kanan foto -> Bagikan (Share) -> Ubah akses menjadi Siapa saja yang memiliki link (Anyone with the link).
  3. Salin link tersebut.
  4. Tempel (Paste) link ke kolom "Link Foto Google Drive" di aplikasi ini. Aplikasi otomatis menampilkan gambarnya.

4. Backup Data

Aplikasi menyimpan data di browser Anda. Wajib lakukan Backup Data secara berkala (tombol di menu kiri bawah) agar data tidak hilang jika browser dibersihkan (Clear Cache).
